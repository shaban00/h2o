Export Customization Tips
We designed this opencasebook.org export using MS Word Styles to simplify customization. The "Styles Pane" in Word's "Home" tab shows a list of styles in the document, and the active style of your current element or selection.
For example, you can quickly change the case text font for the whole document by modifying the "Case Body" style. The style modification screen also shows which style your style is "Based On." If you modify the style listed there, all styles based on it will also change.
